# Khep

**箧** (box, chest), or **K**nowledge, **H**euristics, and **E**ngineering **P**rompts. 

Yixuan's personal collection of agentic coding prompts.

## Installation

If you have `uv` installed, you can use the following install script to install all skills defined in this repo to `$HOME/.claude/skills`.

```bash
curl -fsSL https://raw.githubusercontent.com/Yixuan-Wang/khep/refs/heads/main/install.py | uv run -
```
